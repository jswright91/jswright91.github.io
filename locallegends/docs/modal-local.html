Purpose

The bootstrap-modal-local.js library let you show in a modal box a part of content of your page.

Code

Add the attribute data-local="#myContent" to the button and it will detach the content of the #myContent element to #myModal.

<button id="myButton" type="button" class="btn btn-success" data-toggle="modal" data-target="#myModal" data-local="#myContent">
	<i class="icon-resize-full icon-white"></i>
	<span>View the following content in modal</span>
</button>

<div id="myContent"></div>

<div id="myModal" class="modal hide fade" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
	<div class="modal-header">
		<button type="button" class="close" data-dismiss="modal" aria-hidden="true">&times;</button>
		<h3>Modal header</h3>
	</div>
	<div class="modal-body">
		<p>One fine body…</p>
	</div>
	<div class="modal-footer">
		<a href="#" class="btn" data-dismiss="modal">Close</a>
	</div>
</div>
